Nymph

An object relational mapper for PHP and JavaScript.

Object Relational Mapping in PHP and JavaScript

Nymph lets you create, manipulate, save, and delete data objects from PHP and/or JavaScript without ever writing any SQL, even to create tables. Using Nymph, you can spend more time coding logic, and less time coding data access.

Speedy

Nymph is lean and fast. Not just in processing, but Nymph can be used to prototype in no time at all. You don't have to set up your database, or a schema, and the speed of access is comparable to if you did.

Open Source

Nymph is licensed under the Lesser General Public License. It can be used in open source and proprietary projects.

 

Nymph on GitHub

Easy

Nymph requires zero database knowledge. You can set up a Nymph database and REST endpoint in the time it takes to brew a cup of coffee, and create your first Nymph class in the time it takes to drink it.

Feature Packed in a Small Size

At less than 200KB, Nymph provides a ton of features, giving you the power you need for complex data queries and fast data access.

Out of the box features are best in class, at a fraction of the size of many competitors

  • JavaScript promises for asynchronous access and easy error handling.
  • Easy query syntax makes complex queries much more readable than equivalent SQL.
  • Equivalent PHP and JavaScript API makes coding for backend and frontend easy.
  • On demand loading keeps database queries to a minimum.
  • RequireJS and RequirePHP support allows dependency injection and service location.
  • Horizontally scalable, for enormous databases.
